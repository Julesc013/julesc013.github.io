# julesc013.github.io

Static homepage source for `julesc013.github.io`, built with Eleventy.

## Commands

- `npm run build` builds the site into `_site/`
- `npm run parity:check` verifies the generated output against the canonical `src/` content and assets
- `npm run build:parity` runs the full build plus verification gate
- `npm run verify` aliases `build:parity`
